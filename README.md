## What is this?

## The $111 Lesson

AI agents created 6 Reddit Ads campaigns using browser automation. They chose "Free-form" post type — which has no click-through URL. $111 spent, 76K impressions, zero possible conversions. 

Lesson: Speed without verification is expensive failure. Always verify the atomic unit before scaling.
